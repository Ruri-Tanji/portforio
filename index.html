＜使用技術＞

レポート：Googleドキュメント/Excel

作業の効率化を図る

＜制作期間＞

1か月

＜使用技術＞

HTML/CSS/JavaScript/Python/Adobe illustrator/Adobe XD

＜成果物＞

GitHub

＜詳細＞

誰でも簡単にタスク管理ができるアプリを作成しました。 Djangoのユーザー認証を用いて作成し、Sqlite3でDB管理をしています。

＜コンセプト＞

親子で日本文化を楽しむ。受け継ぐ。

＜制作期間＞

2か月

＜使用技術＞

HTML/CSS/JavaScript /Adobe illustrator/Adobe XD

＜成果物＞

サイト,　 GitHub

＜詳細＞

親子で日本文化(折り紙)を楽しむサイトを作成しました。 クイズや豆知識から、折り方のページに飛べるようにし、子供も飽きずに楽しめるよう工夫しました。 また、全てにフリガナを付け、大人も子供も読みやすいようにしました。

＜コメント＞

ふりがなをふるのが大変でした。ruby要素でフリガナをふるさい、 漢字とふりがなの対応(場所)がうまくいかない箇所がありました。漢字それぞれで分けて行うことで 解決しました。

＜今後＞

豆知識の結果選択の際、一度押したらその設問においては 他の選択肢を押せないようにする。(どれかが押されたらjsを呼ばないようにする。)
